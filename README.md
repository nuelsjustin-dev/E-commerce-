# E-Commerce Website

## 📌 Description
A fully functional e-commerce website built for a retail business to sell products online.
It allows users to browse products, add to cart, and complete purchases
through a database-driven backend system.

## ✨ Features
- Responsive design (mobile and desktop)
- Product listings with categories
- Shopping cart functionality
- Database-driven product management
- AJAX-powered dynamic updates
- Contact form
- Smooth scroll and animations

## 🚀 How to Run Locally
1. Clone the repository:
   git clone https://github.com/nuelsjustin-dev/e-commerce.git
2. Import the MySQL database from the /database folder
3. Configure your database credentials in config.php
4. Run on a local server (XAMPP or WAMP)
5. Open index.html in your browser
